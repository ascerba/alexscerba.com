Site Overview

My name is Alex Scerba, and I am an upcoming transportation design student at the College for Creative Studies in Detroit, MI. I made this website as an exercise in basic web development and hosting. The goal of this project is to create an internet presence not largely controlled by other companies and to learn about the infrastructure of websites.

At the moment, the site is pure HTML and CSS, completely hand written, with no client-side or server-side scripting and is hosted locally. EJS is to be implemented at some point to make maintaining and adding new pages easier. More JavaScript will also be added later to make navigation on mobile devices easier.

Please send any questions or issues to my email: a.scerba02@gmail.com

I plan to move the source for the site to GitHub where it will be easier to submit issues, but for now, and at this scale, email is the easiest option for me.
